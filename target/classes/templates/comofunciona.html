Como Funciona?

Aprenda a navegar por nosso website e conheça o que ele lhe oferece!

Utilize a barra de busca encontrada no início do site para encontrar o que procura.



Utilize o menu de navegação encontrado na parte superior do Website para acessar rapidamente funcionalidades de seu interesse.



Utilize a barra de busca de cor laranja, encontrada no início do Website para encontrar o os serviços que procura.



Na tela de serviços, é possível visualizar todos os serviços.



Utilize a barra de pesquisa encontrada na parte de Serviços do Website para pesquisar por serviços ou prestadores de serviço de seu interesse.



Utilize a opção Solicitar Serviço para fazer uma solicitação do serviço desejado de determinado prestador de serviço.



Utilize essas opções encontradas na tela principal do site para fazer seu cadastro como cliente ou prestador de serviço.



Utilize pelo menu principal a função entrar, para fazer login no site.





Na tela minha conta é permitido a visualização dos dados pessoais e funções como : alterar e excluir informação.





No perfil do prestador, é possível visualizar o estado de todos serviços. Tais como: solicitados, contratados e realizados



No canto superior direito do perfil do prestador (na tela de serviços divulgados), é disponibilizado o botão para inserir novo serviço.



No perfil do cliente é possível visualizar todos os serviços que ele contratou, tal como o avaliar.



E todos os serviços que foram solicitados e/ou que ainda estão em modo de espera.



Utilize a opção Calculadora de materiais para obter os cálculos necessários para sua obra.



Utilize o rodapé encontrado na parte inferior do Website para acessar rapidamente algumas funcionalidades e conhecer mais sobre o Conserta Aqui.








Sobre

Site destinado a trabalhadores do setor de construção civil e pequenos reparos em residências, prédios. O site disponibiliza a oportunidade aos servidores a divulgar seus serviços através de uma ferramenta online de forma prática,simples e gratuita. Além de ser ideal para clientes, que desejam pesquisar preços e ter qualidade em sua obra requerida.
